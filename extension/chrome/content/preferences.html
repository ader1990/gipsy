Setting the route optimizer

GiPSy supports integration with a route optimizer which allows you to directly find out the route that you have flown.

Configuration

Download the optimizer for the appropriate platform from this page. Then go to preferences of Gipsy and set the User command the following way:
  • Set title to some descriptive name
  • Enter the path to the downloaded HSpoints binary
  • Enter the league to optimize as a parameter - e.g. XC for World Xcontest (see bellow for more)
  • Set the type of the program to 'HSPoints'
On Linux/MacOS X systems, start the following command from terminal:
  • chmod +x <name_of_the_points_binary>
Open GiPSy again, right-click (or Ctrl-click on Mac) on a flight and run the user command. When the command finishes you will see the results on the map (you must have Route calculation map layer enabled).
