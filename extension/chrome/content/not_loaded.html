Error loading GPS component

This error message means that an error occured while loading a platform-dependent GPS component.

This can happen when:

  • Your platform is unsupported. Currently supported platform is Linux-i386 with libstdc++.so.6 compiled binary and Windows platform.
  • You share your profile with Windows version of Firefox. In such case run 'firefox -profilemanager', create new profile and install the extension.
  • You are using Windows XP Home system without msvcrt80.dll library. Install the following package: Microsoft Visual C++ 2005 Redistributable Package (x86), then uninstall GiPSy and install it again.
